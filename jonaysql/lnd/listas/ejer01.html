HTTP
Protocolo de Transferencia de Hipertexto, utilizado para transmitir datos en la World Wide Web.
HTML
Lenguaje de Marcado de Hipertexto, estándar para crear y estructurar páginas web.
URL
Localizador Uniforme de Recursos, dirección única que identifica recursos en internet.
TCP/IP
Conjunto de protocolos (Transmisión Controlada/Protocolo de Internet) que gobiernan la comunicación en redes e internet.
Internet
Red global descentralizada que interconecta dispositivos usando protocolos comunes como TCP/IP.
W3C
Consorcio de la World Wide Web, organización que desarrolla estándares y pautas para la web.
Mozilla Foundation
Organización sin fines de lucro que impulsa proyectos de código abierto como Firefox, enfocada en una web accesible y segura.
